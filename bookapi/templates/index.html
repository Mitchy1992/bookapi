This is a simple API made via Django REST framework which allows full CRUD operations on books.

Available Endpoints:

Method Endpoint Description
GET /api/books/ List all books
POST /api/books/ Create a new book
GET /api/books/<id>/ Get details of a book
PUT /api/books/<id>/ Update a book
PATCH /api/books/<id>/ Partially update a book
DELETE /api/books/<id>/ Delete a book

Use tools like Postman or curl to interact with the endpoints:

- Postman : What is postman?

- Curl :

# To GET books
curl -X GET http://localhost:8000/api/books/

# To add books via POST
curl -X POST http://localhost:8000/api/books/ \
-H "Content-Type: application/json" \
-d '{"title": "The Great Gatsby", "author": "F. Scott Fitzgerald", "isbn": "978-0-7432-7356-5", "published_date": "2007-02-09"}'
